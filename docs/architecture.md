# Архитектура конфигурации «Учёт в детейлинг-центре»

## Общая схема

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                         УРОВЕНЬ ПРЕДСТАВЛЕНИЯ                                │
│   ФормаДокумента (ЗаписьНаУслугу)   ФормаДокумента (ОказаниеУслуги)          │
│   ФормаЭлемента (Клиенты)           Форма отчёта (СКД)                       │
└────────────────────────────┬─────────────────────────────────────────────────┘
                             │
┌────────────────────────────▼─────────────────────────────────────────────────┐
│                         УРОВЕНЬ БИЗНЕС-ЛОГИКИ                                │
│                                                                              │
│  ОбщийМодульСервер              ОбщийМодульКлиентСервер                      │
│  ├─ ПолучитьЦенуУслуги()        ├─ РассчитатьСуммуСтроки()                   │
│  └─ ПроверитьОстатокМатериала() └─ ПроверитьЗаполнениеРеквизитов()           │
│                                                                              │
│  Модули документов:                                                          │
│  ├─ ЗаписьНаУслугу.ОбработкаПроведения  → движения в Выручка                │
│  ├─ ОказаниеУслуги.ОбработкаПроведения  → движения в Выручка+ОстаткиМат.    │
│  └─ ПоступлениеМатериалов.ОбработкаПроведения → движения в ОстаткиМат.      │
└────────────────────────────┬─────────────────────────────────────────────────┘
                             │
┌────────────────────────────▼─────────────────────────────────────────────────┐
│                         УРОВЕНЬ ДАННЫХ                                       │
│                                                                              │
│  СПРАВОЧНИКИ           ДОКУМЕНТЫ              РЕГИСТРЫ                       │
│  ┌─────────────┐       ┌───────────────────┐  ┌──────────────────────┐       │
│  │ Клиенты     │──────▶│ ЗаписьНаУслугу    │─▶│ Выручка (обороты)    │       │
│  │ Услуги      │──────▶│ ОказаниеУслуги    │─▶│ ОстаткиМатериалов    │       │
│  │ Материалы   │──────▶│ ПоступлениеМат.   │  │   (остатки)          │       │
│  │ Сотрудники  │       └───────────────────┘  └──────────────────────┘       │
│  └─────────────┘                              ┌──────────────────────┐       │
│                         ПЕРЕЧИСЛЕНИЯ          │ ЦеныУслуг            │       │
│  ┌──────────────────┐   ┌──────────────────┐  │   (периодический)    │       │
│  │ КатегорииУслуг   │   │ СтатусыЗаписей   │  └──────────────────────┘       │
│  └──────────────────┘   └──────────────────┘                                │
└──────────────────────────────────────────────────────────────────────────────┘
```

## Связи между объектами

### ЗаписьНаУслугу

```
Справочник.Клиенты ──────────────────▶ ЗаписьНаУслугу.Клиент
Справочник.Сотрудники ───────────────▶ ЗаписьНаУслугу.Сотрудник
Перечисление.СтатусыЗаписей ─────────▶ ЗаписьНаУслугу.Статус
  ТЧ Услуги:
    Справочник.Услуги ──────────────▶ ЗаписьНаУслугу.Услуги.Услуга
    РегСведений.ЦеныУслуг (срез) ──▶ ЗаписьНаУслугу.Услуги.Цена (авто)
  При проведении:
    ──▶ РегНакопления.Выручка (движение)
```

### ОказаниеУслуги

```
Справочник.Клиенты ──────────────────▶ ОказаниеУслуги.Клиент
Справочник.Сотрудники ───────────────▶ ОказаниеУслуги.Сотрудник
Документ.ЗаписьНаУслугу ────────────▶ ОказаниеУслуги.Основание
  ТЧ Услуги:
    Справочник.Услуги ──────────────▶ ОказаниеУслуги.Услуги.Услуга
  ТЧ Материалы:
    Справочник.Материалы ───────────▶ ОказаниеУслуги.Материалы.Материал
  При проведении:
    ──▶ РегНакопления.Выручка (движение-приход)
    ──▶ РегНакопления.ОстаткиМатериалов (движение-расход)
    ◀── РегНакопления.ОстаткиМатериалов (проверка остатка)
```

### ПоступлениеМатериалов

```
  ТЧ Материалы:
    Справочник.Материалы ───────────▶ ПоступлениеМатериалов.Материалы.Материал
  При проведении:
    ──▶ РегНакопления.ОстаткиМатериалов (движение-приход)
```

## Диаграмма потока данных

```
  Клиент обращается
        │
        ▼
  [ЗаписьНаУслугу]
   - Статус: Новая
   - Услуги + цены (из ЦеныУслуг)
        │ подтверждение
        ▼
  [ЗаписьНаУслугу]
   - Статус: ВРаботе / Выполнена
        │ по завершении
        ▼
  [ОказаниеУслуги] ──── (Основание → ЗаписьНаУслугу)
   - Фиксация факта
   - Расход материалов
        │
        ├──▶ [Выручка] (обороты по услугам, клиентам, сотрудникам)
        └──▶ [ОстаткиМатериалов] (уменьшение остатка)

  [ПоступлениеМатериалов]
        └──▶ [ОстаткиМатериалов] (увеличение остатка)

  [ЦеныУслуг] ◀── ввод цен вручную (периодический регистр)
        └──▶ используется в формах документов для подстановки цены
```
