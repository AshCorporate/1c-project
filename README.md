# 1С: Учёт в детейлинг-центре

Конфигурация для платформы **1С:Предприятие 8.3**, предназначенная для автоматизации учёта услуг, клиентов, материалов и финансовых показателей детейлинг-центра.

---

## Описание

Конфигурация позволяет вести полноценный оперативный учёт в детейлинг-центре:

- **Запись клиентов на услуги** с указанием сотрудника, даты и состава работ
- **Оформление факта оказания услуги** со списанием расходных материалов
- **Поступление материалов** на склад с отслеживанием остатков
- **Ценообразование** — ведение актуальных цен по каждой услуге в разрезе дат
- **Отчётность** — выручка за период и текущие остатки материалов

## Объекты конфигурации

### Справочники

| Справочник   | Назначение                                   | Ключевые реквизиты                              |
|--------------|----------------------------------------------|-------------------------------------------------|
| Клиенты      | База клиентов детейлинг-центра               | Телефон, Email, Автомобиль, ГосНомер            |
| Услуги       | Перечень оказываемых услуг                   | Категория, ВремяВыполненияМинут, Описание        |
| Материалы    | Расходные материалы                          | ЕдиницаИзмерения, Артикул                       |
| Сотрудники   | Мастера и персонал центра                    | Должность, Телефон                              |

### Документы

| Документ               | Назначение                                     | Движения                        |
|------------------------|------------------------------------------------|---------------------------------|
| ЗаписьНаУслугу         | Запись клиента, состав услуг, статус           | Выручка (план)                  |
| ОказаниеУслуги         | Факт оказания услуги, расход материалов        | Выручка + ОстаткиМатериалов     |
| ПоступлениеМатериалов  | Приход материалов на склад                     | ОстаткиМатериалов (приход)      |

### Регистры накопления

| Регистр            | Вид        | Измерения                         | Ресурсы              |
|--------------------|------------|-----------------------------------|----------------------|
| Выручка            | Обороты    | Услуга, Клиент, Сотрудник         | Сумма, Количество    |
| ОстаткиМатериалов  | Остатки    | Материал                          | Количество           |

### Регистры сведений

| Регистр     | Тип        | Измерения | Ресурсы |
|-------------|------------|-----------|---------|
| ЦеныУслуг   | Периодич.  | Услуга    | Цена    |

### Отчёты (СКД)

| Отчёт               | Источник данных              | Группировки                    |
|---------------------|------------------------------|--------------------------------|
| ВыручкаЗаПериод     | РН.Выручка (обороты)         | Услуга, Клиент, Сотрудник      |
| ОстаткиМатериалов   | РН.ОстаткиМатериалов (остат.)| Материал                       |

## Бизнес-логика

1. **Создание клиента** — обязательные поля: Наименование, Телефон
2. **Запись на услугу** — выбор клиента, сотрудника, дата, табличная часть «Услуги» с автоматическим подбором цены из регистра `ЦеныУслуг`
3. **Оказание услуги** — основание — документ «ЗаписьНаУслугу»; при проведении проверяется наличие материалов на складе, затем формируются движения в регистры
4. **Поступление материалов** — пополнение остатков склада
5. **Ценообразование** — цены ведутся в периодическом регистре сведений `ЦеныУслуг`, срез последних на дату документа
6. **Отчёты** — строятся средствами СКД с параметрами периода

## Развёртывание

### Требования
- 1С:Предприятие 8.3.x (рекомендуется 8.3.18+)
- Режим управляемых форм

### Загрузка конфигурации

1. Откройте **Конфигуратор** (Configurator) вашей информационной базы
2. Меню **Конфигурация → Загрузить конфигурацию из файлов...**
3. Укажите путь к директории `src/Configuration/`
4. После загрузки выполните **Конфигурация → Обновить конфигурацию базы данных** (F7)
5. Запустите приложение в режиме **1С:Предприятие**

### Начальное заполнение

1. Заполните справочники: **Сотрудники → Клиенты → Услуги → Материалы**
2. Установите цены в регистре сведений **ЦеныУслуг**
3. Оформите **ПоступлениеМатериалов** для начального остатка склада
4. Начните создавать документы **ЗаписьНаУслугу**
